# Limitations

- alpha software
- `ffmpeg` and `ffprobe` are required for core media workflow
- `faster-whisper` is optional but required for real local transcription
- the real provider adapter is local Ollama only
- no cloud provider execution yet
- no DaVinci Resolve or Premiere integration yet
- no web UI
- no automatic destructive edits
- watch mode is polling-based
- local model output quality depends on the selected local model
