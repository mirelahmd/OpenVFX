# Roadmap

1. Alpha release hygiene and schema stability.
2. Better verification review and export provenance.
3. Deeper local Ollama request/review tooling.
4. Future provider adapters beyond Ollama.
5. Future NLE/timeline packaging artifacts.

Non-goals for the current alpha:

- web UI
- cloud provider execution
- automatic destructive cleanup
- NLE integration
